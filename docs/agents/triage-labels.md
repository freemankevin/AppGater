# Triage Labels

技能使用五个标准分类角色。本文件将这些角色映射到本仓库问题追踪器实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本仓库的标签 | 含义 |
| -------------------------- | ----------- | ---- |
| `needs-triage` | `needs-triage` | 需要维护者评估此问题 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完全明确，AI 代理可直接处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不处理 |

当技能提到某个角色（例如"应用 AFK-ready 分类标签"），使用本表对应的标签字符串。

如需调整，编辑右侧列以匹配实际使用的词汇。
